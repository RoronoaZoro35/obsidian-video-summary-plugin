# 基于文件的缓存系统

## 🚀 概述

视频总结插件现在使用基于文件的缓存系统，替代了之前的localStorage方案。新系统提供了更好的性能、更大的容量和更强的可靠性。

## 📁 目录结构

```
.obsidian/plugins/video-summary-plugin/
├── cache/
│   ├── index.json          # 缓存索引文件
│   └── items/              # 缓存条目目录
│       ├── abc123.json     # 单个缓存项（文件名是URL的哈希值）
│       ├── def456.json
│       └── ...
```

## 🔧 核心特性

### 1. **大容量存储**
- 不受浏览器localStorage限制（5-10MB）
- 支持GB级别的缓存存储
- 每个缓存项独立存储

### 2. **自动数据迁移**
- 启动时自动检测旧localStorage数据
- 无缝迁移到新文件系统
- 迁移完成后自动清理旧数据

### 3. **智能索引管理**
- 集中式索引文件管理所有缓存项
- 快速查找和统计
- 自动清理过期项

### 4. **崩溃恢复**
- 单个文件损坏不影响整体系统
- 自动检测和修复损坏的索引项
- 优雅降级处理

## 📊 缓存项结构

### 索引项 (index.json)
```json
{
  "version": "2.0.0",
  "lastUpdated": 1703123456789,
  "maxSize": 1000,
  "expiryDays": 30,
  "items": {
    "https://example.com/video.mp4|summary|zh": {
      "url": "https://example.com/video.mp4",
      "mode": "summary",
      "language": "zh",
      "timestamp": 1703123456789,
      "expiryTimestamp": 1705715456789,
      "size": 2048,
      "path": "cache/items/abc123.json"
    }
  }
}
```

### 缓存内容项 (items/*.json)
```json
{
  "url": "https://example.com/video.mp4",
  "mode": "summary",
  "language": "zh",
  "result": {
    "summary": "视频摘要内容...",
    "video_transcript": "视频转录内容..."
  },
  "timestamp": 1703123456789,
  "expiryTimestamp": 1705715456789
}
```

## 🎯 使用方法

### 1. **查看缓存统计**
- 设置 → 视频总结插件 → 高级配置 → 缓存配置
- 点击 **📊 缓存管理** 按钮
- 查看缓存统计和内容预览

### 2. **管理缓存**
- **测试缓存**: 添加测试数据
- **清理过期**: 删除过期项
- **清空所有**: 完全清空缓存
- **单条删除**: 在预览中删除特定项

### 3. **配置选项**
- **启用缓存**: 开启/关闭缓存功能
- **缓存过期时间**: 设置缓存有效期（天）
- **最大容量**: 自动管理缓存项数量

## 🔄 数据迁移

### 自动迁移流程
1. 插件启动时检测localStorage中的旧数据
2. 逐个读取并验证旧缓存项
3. 转换为新格式并保存到文件系统
4. 更新索引文件
5. 清理localStorage中的旧数据
6. 显示迁移结果

### 迁移日志
```
发现旧缓存数据，开始迁移...
迁移完成，共迁移 X 项
```

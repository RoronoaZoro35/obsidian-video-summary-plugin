# 缓存预览功能使用指南

## 概述

缓存预览功能让你可以详细查看和管理视频总结插件的缓存内容，包括统计信息、具体缓存项、过期状态等。

## 功能特点

### 📊 详细统计信息
- **总缓存项**: 当前已缓存的视频处理结果数量
- **最大容量**: 缓存系统的最大容量限制
- **过期项**: 已过期的缓存项数量
- **使用率**: 缓存容量的使用百分比

### 🔍 缓存内容预览
- **URL显示**: 显示缓存的视频链接（自动截断长URL）
- **处理模式**: 显示是"完整总结"还是"仅文稿"
- **语言标识**: 显示处理语言（中文/英文/日文）
- **时间信息**: 显示缓存创建时间（今天/昨天/X天前）
- **过期状态**: 显示缓存项是否已过期

### ⚙️ 管理操作
- **刷新**: 刷新缓存统计和内容
- **清理过期**: 一键清理所有过期缓存项
- **删除单项**: 删除特定的缓存项
- **关闭**: 关闭预览窗口

## 使用方法

### 1. 打开缓存预览
1. 进入Obsidian设置
2. 找到"视频总结插件"设置
3. 在"高级配置" → "缓存配置"部分
4. 点击"查看统计"按钮

### 2. 查看统计信息
预览窗口顶部显示四个统计卡片：
```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│  总缓存项   │  最大容量   │   过期项    │   使用率    │
│     15      │    1000     │     3       │    1.5%     │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

### 3. 浏览缓存内容
每个缓存项显示为卡片形式：
```
┌─────────────────────────────────────────────────────────┐
│ https://bilibili.com/video/BV123456...                 │
│ [完整总结] [中文] [今天]                    [有效] [🗑️] │
└─────────────────────────────────────────────────────────┘
```

### 4. 管理缓存
- **查看状态**: 绿色"有效"表示缓存可用，红色"已过期"表示需要清理
- **删除单项**: 点击🗑️按钮删除特定缓存项
- **批量清理**: 点击"清理过期"按钮清理所有过期项

## 界面说明

### 统计卡片
- **悬停效果**: 鼠标悬停时卡片会轻微上浮
- **颜色编码**: 过期项数量用红色显示
- **响应式布局**: 自动适应不同屏幕尺寸

### 缓存项列表
- **滚动支持**: 超过10项时显示滚动条
- **悬停高亮**: 鼠标悬停时高亮显示
- **状态标识**: 清晰显示有效/过期状态
- **操作按钮**: 每个项目都有删除按钮

### 操作按钮
- **刷新**: 重新加载缓存数据
- **清理过期**: 清理过期缓存（仅在有过期项时显示）
- **关闭**: 关闭预览窗口

## 使用场景

### 1. 监控缓存使用
- 定期查看缓存统计，了解使用情况
- 监控缓存使用率，避免接近容量上限
- 检查过期项数量，及时清理

### 2. 调试问题
- 查看特定视频是否被缓存
- 检查缓存项的过期状态
- 验证缓存配置是否正确

### 3. 管理缓存
- 删除不需要的缓存项
- 清理过期缓存释放空间
- 维护缓存数据的质量

## 注意事项

### 1. 缓存项限制
- 最多显示前10项，避免界面过长
- 超过10项时显示"还有X项未显示..."提示
- 使用滚动条浏览更多内容

### 2. 删除操作
- 删除操作不可撤销，请谨慎操作
- 删除后立即从界面移除
- 删除操作会同步到localStorage

### 3. 性能考虑
- 大量缓存项时，预览加载可能需要一些时间
- 建议定期清理过期缓存，保持性能
- 缓存统计实时更新，无需手动刷新

## 故障排除

### 1. 无法显示缓存内容
- 检查是否启用了缓存功能
- 确认localStorage权限正常
- 查看控制台是否有错误信息

### 2. 统计信息不准确
- 点击"刷新"按钮重新加载
- 检查缓存配置是否正确
- 确认插件版本是最新的

### 3. 删除操作失败
- 检查浏览器localStorage权限
- 确认缓存数据格式正确
- 尝试刷新后重新操作

## 最佳实践

### 1. 定期检查
- 建议每周检查一次缓存状态
- 关注过期项数量，及时清理
- 监控缓存使用率变化

### 2. 合理配置
- 根据使用频率设置合适的过期时间
- 避免设置过短的过期时间
- 根据设备内存调整缓存容量

### 3. 有效管理
- 定期清理过期缓存
- 删除不再需要的缓存项
- 保持缓存数据的时效性

缓存预览功能提供了完整的缓存管理体验，让你可以：
- **全面了解**: 查看详细的缓存统计和内容
- **有效管理**: 清理过期项，删除不需要的缓存
- **实时监控**: 监控缓存使用情况和状态
- **问题诊断**: 调试缓存相关的问题

通过合理使用这个功能，你可以更好地管理视频总结插件的缓存，确保系统性能和用户体验的最佳状态。
